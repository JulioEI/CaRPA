This folder contains the example codes and datasets for simulating the one-photon movie from two-photon volumetric movies.
Please refer to the "image_simulation_pipeline.pdf" file for more details.
The full two-photon datasets can be obtained from: Kim, Tony Hyun, et al. Cell Reports 17.12 (2016): 3385-3394.
The example simulation result can be downloaded from: https://drive.google.com/open?id=0B7IMqq5UNWsMVDlwakZjczRNcTA
